Process

  1. Pencil Drawing

    2. I start with a pencil drawing because I'm awful at using a tablet. You can really see the differences between the initial drawing and the final product.
    (Don't tell anyone, but this entire project was just an excuse to draw leg warmers.)

  2. Line Art

    3. I make outlines of all of the parts I think need their own layers using vector tools.

  3. Color Art

    4. Now I work on the general color theme. I took the colors straight from Avery's reference sheet by @tictactuh_ on Twitter. I also made some small adjustments, including making Avery's head smaller, because I couldn't stop thinking "why the long face?"

  4. Making it Move

    5. We make it move and we make it loop. Use lots of ease in out. She's floating now, and having a lot of fun.

  5. Cool Line Stuff

    6. I've never done something like this before, so feedback is greatly appreciated. I colored the lines based on the color art, and I also tapered the lines. I think it looks cool.

  6. Finalizing

    7. I added a nice border, gave her some glow and some color cards to the right, and put some Average4k gameplay in the back. I took the footage from the Avg4k discord server, if you can find it, you get a cookie! (no promises though)
